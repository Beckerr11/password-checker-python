# 🔐 Password Checker Python

Projeto que verifica a força de uma senha.

## Regras

A senha é analisada considerando:

- tamanho mínimo
- letras maiúsculas
- números
- caracteres especiais

## Tecnologias

- Python
- Regex

## Como executar

Execute:

```
python main.py
```
